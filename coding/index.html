TLVR

TLVR (short for Tetralingual Voice Record) is a website where people can browse text and audio files from the game Arknights, in a way where they can easily switch between languages that the game is available in. I built it explicitly to help language-learners and to easily compare between different voice acting.

The design was created in Figma with a responsive mobile-first approach before then adjusted for desktop. The website was built using SvelteKit with intention of being full-stack, although it’s then adapted to a static site for easier access and deployment.

Data for the site’s content was gathered from other community-created databases, collected using Python scripts, and processed into JSON for the site.

Multilingual Genshin Database & Pradewata

Multilingual Genshin Database hosts text data from the video game Genshin Impact, in all the languages it’s available in. It’s built from the grounds-up to make comparison between languages easier, which is hopefully helpful for language-learners, those interested in video game localizations, or just fans of the games.

It was built with Remix. The data was taken from a public repository and then further processed with custom Python scripting.

This database is a sister site to Pradewata, my personal project to retranslate Genshin Impact into Indonesian in a way that retains each character’s personality. The website was built with the same stack.

MiniBlaseball

#MiniBlaseball is a website I was commissioned to build by HetreaSky to display his collection of “mini” art, each one depicting a player or other entity in the (now defunct) live-service absurdist horror sports game, Blaseball.

The website itself is a single-page app (SPA) which reads data from JSON files. It features sorting and filtering by various factors, multiple “pages”, light and dark themes, responsive design, lazy-loading of images, and display of either randomized or specific art, depending on the filter.

It was first built in Riot, a niche UI library, before being rewritten in Svelte. Data was written into each image’s file name and additionally in an Eno file, which was parsed with Python into JSON for use by the web app.

Lightning Cards

Lightning Cards is a flash card web app built with Riot, a niche UI library similar to Vue. The web app works as an SPA that automatically saves data in the browser using the Local Storage API, but also allows importing/exporting this data through JSON.

I built Lightning Cards because I wanted a flash card app that is simple to use, where I can easily and quickly create new cards and then practice with them like I would physical paper cards. I found existing apps like Anki is too complicated for my needs.
