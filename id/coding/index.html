CSS wizard, HTML wrangler, JavaScript tamer

Aku pertama membangun website saat masih bocah umur sembilan tahun dan sampai sekarang entah kenapa masih juga seneng ngoding. Tujuan utamaku adalah membangun website yang unik, menyenangkan untuk dilihat, dan, yang paling penting, bisa menyajikan informasi secara aksesibel. Musuh bebuyutanku adalah perintilan yang membuat website jadi berat dan sulit diakses.

Kemampuanku berat sebelah di ranah frontend. CSS dan HTML modern sudah kuanggap master-level, dan aku juga cukup mahir JavaScript. Framework favoritku Svelte, tapi aku juga sudah pernah membangun dengan React dan Vue. Di luar frontend, aku juga berpengalaman dengan Python, Ruby, dan Lua, yang kugunakan untuk di antaranya otomatisasi, visualisasi data, web backend, dan sedikit-sedikit gamedev.

Aku masih belum begitu mahir desain visual, tapi ya, sedikit demi sedikit lama-lama menjadi bukit!

Saat ini aku buka jasa membangun website kecil. Kalau kamu tertarik membangun website unik untuk produk, bisnis, atau hobimu, silakan kontak aku!

Di bawah ini adalah beberapa proyek-proyekku:

Website ini!

Website ini dibangun dengan Eleventy dan dihosting oleh GitHub Pages. Desainnya pertama divisualisasikan di Figma, lalu kodingnya dilakukan dengan HTML/CSS vanilla dengan beberapa markup Nunjucks. Gambar ikon ditampilkan dengan IcoMoon.

Versi lama dari website ini dibuat dengan static site generator buatanku sendiri, Plater, yang dibangun dengan Python. Akan tetapi proyek ini sudah tidak lagi dirawat.

MiniBlaseball

MiniBlaseball adalah website yang kubangun untuk @HetreaSky untuk menjadi galeri seri gambar "mini" buatannya. Setiap "mini" menampilkan karakter dalam Internet League Blaseball. Website ini dibangun dengan Svelte dan memiliki fitur-fitur di antaranya: filter dan pengganti urutan, pencarian cepat, lazy-loading untuk gambar, perubahan tema terang/gelap secara instan, desain responsif, dan ukuran gambar yang adaptif.

Big Black

Big Black adalah template untuk blog yang didesain agar responsif terhadap ukuran layar, menggunakan ukuran teks yang besar, dan berdasarkan pada konsep brutalis. Template ini dibangun langsung dengan CSS dan HTML modern tanpa framework atau library eksternal.

Lightning Cards

Lightning Cards is a simple flash card web app that can help you memorize thing. It runs entirely on the front end and built with Riot. I still use to help me practice foreign languages, although I've been thinking up rebuilding and upgrading it for a while.

No-Nonsense Basic JavaScript Guide for the Novice Web Developer
