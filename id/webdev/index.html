webdev

CSS wizard, HTML wrangler, JavaScript tamer

Aku pertama membangun website saat masih bocah umur sembilan tahun dan sampai sekarang entah kenapa masih juga seneng ngoding. Prinsip dasarku saat membangun website adalah berusaha membuat sesuatu yang unik, menyenangkan untuk dilihat, dan, yang paling penting, bisa menyajikan informasi secara aksesibel. Musuh bebuyutanku adalah perintilan yang membuat website jadi berat dan sulit diakses.

Kemampuanku berat sebelah di ranah frontend. CSS dan HTML modern sudah kuanggap master-level, dan aku juga cukup mahir JavaScript. Framework favoritku Svelte, tapi aku juga sudah pernah membangun dengan React dan Vue. Di luar frontend, aku juga berpengalaman dengan Python, Ruby, dan Lua, yang kugunakan untuk, di antaranya, otomatisasi, visualisasi data, web backend, dan sedikit-sedikit gamedev.

Aku masih belum begitu mahir desain visual, tapi ya, sedikit demi sedikit lama-lama menjadi bukit!

Saat ini aku sudah punya pekerjaan full-time, tapi kalau kamu tertarik membangun website unik untuk produk, bisnis, atau hobimu, silakan kontak dan mungkin aku masih bisa membantu!

Di bawah ini adalah beberapa proyek-proyekku:

Multilingual Genshin Database

Multilingual Genshin Database menyimpan data teks dari game Genshin Impact dalam semua bahasa yang ada untuk game ini. Database ini dari awal dibangun agar mempermudah perbandingan antar versi bahasa, dan harapannya bisa membantu bagi mereka yang sedang belajar bahasa, yang tertarik dengan penerjemahan game, dan juga penggemar-penggemar Genshin secara ummum.

Website ini dibangun dengan Remix. Data diambil dari repositori publik yang kemudian diproses lagi dengan scripting custom lewat Python.

Pradewata

Pradewata adalah proyek pribadi untuk menerjemahkan sebagian Genshin Impact ke bahasa Indonesia yang, harapannya, lebih mencerminkan tiap karakter. Website ini adalah saudara Multilingual Genshin Database; keduanya menggunakan teknologi yang sama.

Website ini!

Website ini dibangun dengan Eleventy dan dihosting oleh GitHub Pages. Desainnya pertama divisualisasikan di Figma, lalu kodingnya dilakukan dengan HTML/CSS vanilla dengan beberapa markup Nunjucks. Gambar ikon ditampilkan dengan IcoMoon.

Versi lama dari website ini dibuat dengan static site generator buatanku sendiri, Plater, yang dibangun dengan Python. Akan tetapi proyek ini sudah tidak lagi dirawat.

MiniBlaseball

MiniBlaseball adalah website yang kubangun untuk @HetreaSky untuk menjadi galeri seri gambar "mini" buatannya. Setiap "mini" menampilkan karakter dalam Internet League Blaseball. Website ini dibangun dengan Svelte dan memiliki fitur-fitur di antaranya: filter dan pengganti urutan, pencarian cepat, lazy-loading untuk gambar, perubahan tema terang/gelap secara instan, desain responsif, dan ukuran gambar yang adaptif.

Big Black

Big Black adalah template untuk blog yang didesain agar responsif terhadap ukuran layar, menggunakan ukuran teks yang besar, dan berdasarkan pada konsep brutalis. Template ini dibangun langsung dengan CSS dan HTML modern tanpa framework atau library eksternal.

Lightning Cards

Lightning Cards adalah aplikasi web flash card sederhana yang kubangun untuk membantuku menhafal kata. Aplikasi ini seluruhnya berjalan di frontend dan dibangun dengan Riot. Sampai sekarang,aplikasi ini masih kugunakan untuk berlatih belajar bahasa, tapi aku juga terpikir untuk membangun ulang dan menambah fitur-fitur lagi.

No-Nonsense Basic JavaScript Guide for the Novice Web Developer

Judul yang super panjang ini adalah tajuk untuk buku petunjuk yang kutulis untuk membantu web developer pemula belajar JavaScript dasar. "Buku" ini kutulis sedemikian rupa untuk membantu orang-orang yang cara pikirnya seperti diriku saat aku masih pemula.
