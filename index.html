Coding

Programming is hell but it's also one of the few things I can be kind of good at, or at least has a lot of passion in doing.

I mostly program in Python, but I also do a bit C++. I do front-end web development with vanilla HTML, CSS, and jQuery, with some basic Javascript. When I'm feeling masochist, I enjoy diving through a variety of web application frameworks.

I also make games with RenPy and Twine although so far there's nothing solid yet.

I use Windows 7 for my programming shenanigans (yes, I use Windows, have mercy), utilising Notepad++ and its crappy-but-still-functional Command Line. For my writing work, I always strive to use plaintext + Markdown first, formatting later.
