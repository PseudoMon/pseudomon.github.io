CSS wizard, HTML wrangler, and JavaScript tamer

I started building websites when I was nine years old and have not stopped since. I believe that websites can be built to have unique personalities, to be a joy to behold, but most of all to be an accessible way to show information on the web. Bloat is my nemesis.

My skill lies mostly in frontend web development. I have mastery of modern CSS and HTML and a good grasp of JavaScript. My framework of choice is Svelte, but I've also worked with React and Vue. Outside of the frontend, I have experience with Python, Ruby, and C#, which I've used for automation, data visualization, web backend, and game development.

I'm currently gainfully employed and not looking for work!

Below are some of my personal projects!

Multilingual Genshin Database

Multilingual Genshin Database hosts text data from the video game Genshin Impact, in all the languages it's available in. It's built from the grounds-up to make comparison between languages easier, which is hopefully helpful for language-learners, those interested in video game localizations, or just fans of the games.

It was built with Remix. The data was taken from a public repository and then further processed with custom Python scripting.

Pradewata

Pradewata is a personal project to retranslate a portion of Genshin Impact into Indonesian, aiming for a translation that better reflects the characters' personality. It is a sister site to Multilingual Genshin Database and utilizes the same technology.

This website!

This website was built using Eleventy and hosted on GitHub Pages. The design was first visualized on Figma before I then code it by hand into vanilla HTML/CSS sprinkled with Nunjucks markup. Icons is provided by IcoMoon.

A previous version of this website was made using my own static site generator, Plater, which I built with Python. That project is no longer maintained.

MiniBlaseball

MiniBlaseball is a website I built for @HetreaSky to display his series of art depicting the many players in Internet League Blaseball. It was built with Svelte and contain these features, among other: filtering and sorting, quick searching, lazy-loading of images, instanteous change between light and dark themes, responsive design, and adaptive display area.

Big Black

Big Black is a responsive blog theme made with large font size and brutalist concept in mind. It was built by hand with modern CSS and HTML with no external framework or library.

Lightning Cards

Lightning Cards is a simple flash card web app that can help you memorize thing. It runs entirely on the front end and built with Riot. I still use to help me practice foreign languages, although I've been thinking up rebuilding and upgrading it for a while.
